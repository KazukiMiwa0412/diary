# 使用技術
- PHP 7.3.27
- laravel 6.20.29
- MySQL 15.1
- AWS
    - cloud9
    - S3

# 機能一覧
- ユーザー登録、ログイン機能
- 日記作成
    - タイトル、日付、本文入力
    - 画像アップロード機能（選択画像表示）
    - 作成した日記をTwitterにツイートすることができる
 
- 日記編集
    - 各項目の編集
    - 画像の追加、削除機能

- 日記検索機能
